# FrostFoe's Digital Cosmos 🌌

Welcome to FrostFoe's digital cosmos, a hyper-optimized personal blog and portfolio built with the latest web technologies. This project showcases a sleek, high-performance web application crafted using Next.js with the App Router, React, and TypeScript. The UI is styled with Tailwind CSS and features elegant components from ShadCN UI, all brought to life with immersive 3D effects and fluid animations powered by Framer Motion. Key features include a full-fledged blog engine using MDX for rich content, complete with categories, tags, and a lightning-fast search; an interactive particle system on the homepage; and advanced performance techniques like server-side caching, lazy loading, and UI streaming to ensure a blazing-fast experience. The site includes dedicated pages for showcasing projects, a detailed 'About Me' section, and a functional contact form, all wrapped in a clean, futuristic monochrome theme.

## Getting Started

This project was built in Firebase Studio. To get started exploring the code, take a look at `src/app/page.tsx`.
